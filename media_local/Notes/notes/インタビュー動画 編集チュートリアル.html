インタビュー動画 編集チュートリアル



ここでは「GoogleDrive」「Vrew」「Davinci」「YoutubeStudio」を組み合わせた動画編集を一通り経験してもらいます。

Vrewでは、自動で字幕入れと無音部分カットを行い、不要部分のカットなどの構造整理を行います。
Davinciでは、画角を調整し、音や追加テロップなどで動画をデザインします。
YoutubeにUPした後、字幕を含めた情報入力をして完了です。
素材共有には「GoogleDrive」を使います。

さらに、Davinciでは、Brackmagic Cloud を利用してプロジェクトをメンバー同士で共有します。
これにより、「人物別」の作業後、スムーズに「質問別」に移行することができます。
GoogleDrive15GB以上使おうと思うと月250円必要です。それ以外は全て無料です。

注意
一般に「複数カメラ」や「音声別撮り」の編集をしようと思うと、複数ファイルの「同期処理」が必要です。
しかし、Vrewで読み込み込んだ後に行うことができる、良い「同期処理」の方法は(大橋が認知する限り)ありません。
そこで、ここでは、「単一カメラ」での「音声付き動画」の編集を前提とします。

手順の全貌
  1. 環境構築
    1. GoogleDrive環境
      1. GoogleDriveインストール: 参考
      2. 素材フォルダにアクセスし、右クリックで「ドライブにショートカットを作成」
    2. Vrew環境
      1. Vrew公式サイトからダウンロード、インストール
      2. アプリ内から無料会員登録→利用制限が解除される
    3. Davinci環境
      1. 無料版のインストール
      2. Brackmagic Cloud ID作成(右上の雲マークから)
      3. 作成したBrackmagic Cloud ID(登録メールアドレス)をプロジェクトの「オーナー」に伝える。注：「オーナー」は受け取ったIDを使ってプロジェクトメンバーを追加します。
  2. 編集
    1. Vrewで構造整理
      1. Google Driveからデータを選んで、読み込む。参考注：読み込みの際に、データの変換を行う必要がある場合があります。変換ファイルの出力先は、素材と同じフォルダを指定してください。 注：読み込みと同時に、自動文字起こしが完了します。 この時点で誤植を訂正する必要はありません。この字幕はあくまで目印であり、最終的な字幕はYoutubeStudioで作成します。 注：動画ファイルが複数に分かれている場合は、「ホーム」タブ＞動画を追加するから、全ての動画を追加してください。
      2. 無音区間を短縮 「編集」タブ＞「無音区間を短縮」
      3. 不要シーンのカット
        1. 不要クリップを削除する
        2. 話題の切り替えタイミング「空きクリップ」を挿入する【推奨】これは単なる目印です。Davinciに引き継いだ際に消えない目印としては「空きクリップ」を使います。 （おそらく他の要素は引き継ぎがうまくいかない？）
      4. Davinci用にプロジェクトを書き出す 参考
    2. Davinciでデザイン
      1. プロジェクトの複製【推奨】 以前作った他のプロジェクトを一つ選び、複製します。プロジェクトを新規作成しても問題ありませんが、定型質問のテロップやロゴなど、以前作った素材を流用するためにも、複製を推奨します。複製後、プロジェクト名の変更と、不要素材の消去を行いましょう。
      2. タイムラインの読み込み 参考注：「プロジェクトの読み込み」ではなく、「タイムラインの読み込み」です。
      3. 素材データの再リンク 参考ほとんどの場合、データのリンクに赤いエラーが出るので、Google Driveに再リンクさせましょう。 注：再リンクフォルダの選択時に、Google Drive内のショートカット内に入ろうとするとエラーが起きます。ショートカットフォルダの中には入らずに、「マイドライブ」自体を指定してください。
      4. 画角調整【推奨】 かっこいい画角を作ります。一つのクリップで調整してから、全てのクリップに「属性をペースト」します。
      5. 効果つけ
        1. 「空きクリップ」の箇所に質問テロップの挿入 定型質問はドラッグアンドドロップすれば終わり
        2. 効果音を挿入【推奨】
        3. OpeningとEndingの挿入【推奨】
      6. 動画の書き出し字幕(形式としてはTextになってるはず)を無効にして動画を書き出す。参考
        注：質問テロップ以外の字幕は、YoutubeStudioで作成します。
      7. 動画のアップロードYoutubeStudio にログイン。「放課後の教室」に紐づいたアカウントでログインし、「放課後の教室」に切り替える。 画面右上の「作成」＞「動画をアップロード」
  3. YoutubeStudio での情報入力
    1. 字幕をつける。
      1. 自動字幕表示を強制にするキーワード「yt:cc=on」が含まれていることを確認する 参考
      2. 自動生成された字幕をの誤植を訂正する 参考字幕の訂正は、これ以降どんなタイミングでも修正できます。
    2. メタ情報の入力
      1. タイトル、説明を入力する
      2. サムネイルを作ってUPする
