3．2の解決策が見つかった．
中心が一致する2つの同心円で挟まれる部分を表すAnnulus（輪？）を使って
def Ensyuu(ten,radius,stroke_width=0.05,color = WHITE):
return Annulus(
inner_radius = radius-stroke_width, outer_radius=radius,
color = color
).move_to(ten.get_center())
という円周関数を定義した．
関数定義はキレイにいくと楽しい．
途中，陰関数でf(x,y)**2+g(x,y)**2=0として，共通部分を描こうととしたが，
